==路由建立==

- https://   http 協定 (加密與否，尤其表單需要加密)
- youtube.com   domain 網址，主網域
- https://www.youtube.com/search   search = 路徑(path)
- https://www.youtube.com/search?q=n8n&source=unknown   參數與多個參數的連接，參數將被瀏覽器解析為物件格式，用來撈資料庫
